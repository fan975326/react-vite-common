## react-vite

## Describe

基于 vite 的 react 通用模板，项目配置好 eslint 以及 commitlint 等...

## 部分配置了解请查看官网

- commitlint 官网 https://github.com/conventional-changelog/commitlint

- husky（配置 git 提交前的 hooks） 官网 https://github.com/typicode/husky

## Install node_modules

```javascript
yarn;
```

or

```javascript
npm install;
```

## Start Project

```javascript
yarn vite;
```
